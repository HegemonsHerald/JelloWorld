### Method
"Named Sequence of program steps", either associated with a class or an instance of a class.

### Formal and Actual Parameters
Formal parameters are variable declarations in a method definition, whereas actual parameters
are the arguments passed to a function on call -- in place of the formal parameters.
Actual parameters have a value, while formal parameters are merely placeholders for values.

### Type
The kind of data contained in a variable; can either be a class like `String` or a primitive like `int`.

### Scope
The region of code, in which a variable is valid, that is, *it exists/can be used*, is that variable's scope.

### Control Flow Statement
A statement whose side-effect is, that it conditionally executes code and potentially repeats it based on its context at runtime.
